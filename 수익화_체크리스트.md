# 모두도구 — 수익화·유입 체크리스트 (원석씨 아침 클릭용)

라이브: **https://modutools.vercel.app** (2026-07-21 새벽 배포, 도구 10종 전부 작동)
코드: github.com/wws21c/modutools (push = Vercel 자동 재배포 아님 — CLI 배포 방식. 수정 시 클코가 재배포)

## 냉정한 수익 산식 먼저 (기대치 관리)

- 한국 애드센스 콘텐츠 사이트 RPM ≈ 1,000~3,000원 / 1,000PV
- **월 100만원 = 월 40만~100만 PV = 일 1.3만~3.3만 PV**
- 신규 도구 사이트가 여기 도달하는 데 통상 수개월(SEO 숙성), 대부분은 영영 못 감
- 즉 이 사이트는 "당장 100만"이 아니라 **복권 + 방치형 자산**. 서버비 0원이라 하방은 0, 상방은 열려 있음. 판정 장치 = 아래 kill-switch.

## 1단계 — 오늘 아침 (합계 약 30분)

- [ ] **도메인 구입** (연 1.5~2만원, 예: modutools.com / modu.tools / 모두도구.kr) — **애드센스는 vercel.app 같은 무료 서브도메인 사이트를 등록 못 받아줌**. 도메인이 애드센스의 전제조건. Vercel 대시보드 → modutools 프로젝트 → Domains에 연결(SSL 자동)
- [ ] **구글 서치콘솔 등록** + sitemap.xml 제출 (5분)
- [ ] **네이버 서치어드바이저 등록** + sitemap 제출 (5분) — 한국 도구 검색 유입은 네이버가 절반 이상
- [ ] **쿠팡 파트너스 가입** (즉시 승인형) — 승인되면 클코가 도구 페이지에 맥락 맞는 배너 삽입 (예: BMI → 체중계, 이미지압축 → 웹캠)

## 2단계 — 도메인 연결 후 (클코에게 "도메인 샀어"라고만 말하면 됨)

- (클코) canonical·sitemap·robots를 새 도메인으로 일괄 치환 + 재배포
- [ ] **애드센스 신청** (adsense.google.com — 구글 계정으로 사이트 추가, 심사 며칠~2주)
- 승인 후: (클코) `ads.js`의 CLIENT에 ca-pub ID 입력 + ads.txt 생성 + 재배포 → 전 페이지 광고 게재 시작
- 대안·병행: **카카오 애드핏**(심사 상대적으로 관대), 애드센스 거절 시 백업

## 3단계 — 유입 (진짜 게임)

- (클코) 반응속도 테스트가 유일한 바이럴 후보 — 공유 버튼 내장됨. 커뮤니티 공유 문구 초안은 클코가 준비, **게시 여부·계정은 원석씨 판단** (외부 발행이라 선제 실행 안 함)
- (클코) 검색 수요 기반 도구 추가 여지: 전역일 계산기, 평수 변환, 근로소득세 연말정산 모의계산, 글자수 → PDF 등. 서치콘솔 데이터 보고 추가 판단
- 도구 1개당 페이지 1장 = 검색 롱테일 1개. 사이트의 성장 = 도구 수 × 숙성 시간

## kill-switch (제안 — 확정 대기)

- **60일(9/21) 기준 일 500PV 미만이면 도구 추가·시간 투입 중단, 방치 모드 전환** (서버비 0이라 삭제는 안 함 — 롱테일은 계속 굴러감)
- 애드센스 거절 2회 시 애드핏 단독으로 전환

## 구조 메모

- 순수 정적 HTML/CSS/JS, 프레임워크·서버·DB 없음 → 유지비 0원, 트래픽 폭증에도 비용 0
- 모든 도구 클라이언트 사이드 처리 (개인정보 무전송 — 이게 셀링포인트라 각 페이지에 명기)
- 광고는 `ads.js` 한 파일이 전 페이지 제어 — 승인 전엔 아무것도 안 보임
- 연봉 계산기 세액은 간이세액표 근사치(면책 문구 있음). 2027 요율 개정 시 클코가 갱신 필요
